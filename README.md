# Fiable Frontend Assessment

It has Atomic Design principles, SOLID principles, integrates Material UI, includes Storybook for UI testing, and Jest for unit testing.

# Includes

- SOLID Principle
- Material UI
- Atomic Design
- Story Book
- Unit Test / Jest
- Constant Variables
- Utilities

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.

### `npx jest`

Runs the testing 

### `npm run storybook`

Runs the storybook for UI Testing
